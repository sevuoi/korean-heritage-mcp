# MCP Tools

## find_nearby_heritage
- Purpose: Search heritage sites around a location
- Input: latitude, longitude, location, radius_km
- Output: nearby heritage list

## find_heritage_by_designation
- Purpose: Lookup heritage by designation type and number
- Input: designation_type, designation_number
- Output: matching heritage details

## plan_heritage_trip
- Purpose: Build a structured trip plan for a region
- Input: region, days, transport, themes
- Output: itinerary and notes
