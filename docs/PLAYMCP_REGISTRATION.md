## Remote MCP server conditions
- HTTPS endpoint is recommended for public deployment
- Endpoint should expose the MCP streamable HTTP path
- The server name should be short and descriptive
- Tool annotations explicitly set title, read-only, destructive, open-world, and idempotent hints
